# env-config

[![npm version](https://img.shields.io/npm/v/env-config.svg)](https://www.npmjs.com/package/env-config)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Environment variable reader with defaults and type casting

## Installation

```bash
npm install env-config
```

## Functions

- `getEnv` — Get env var with optional default
- `getEnvInt` — Get env var as integer
- `getEnvBool` — Get env var as boolean
